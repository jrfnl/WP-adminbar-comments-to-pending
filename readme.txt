Changes the link from the Adminbar comments bubble to go straight to the 'Pending' comments queue.

== Description ==

Tiny plugin which changes the link from the Adminbar comments bubble to go to the 'Pending' comments queue instead of to the 'All' comments overview page.

That's all ;-)

== Installation ==

1. Extract the .zip file for this plugin and upload its contents to the `/wp-content/plugins/` directory. Alternatively, you can install directly from the Plugin directory within your WordPress Install.
1. Activate the plugin through the "Plugins" menu in WordPress.


== Screenshots ==
